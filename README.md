A 2D/top-down mini racing game where players race against AI-generated rivals powered by MiMo AI.

[![Deploy to Netlify](https://www.netlify.com/img/deploy/button.svg)](https://app.netlify.com/start/deploy?repository=https://github.com/senaburayut456/mimo-street-racer)

## Features

- **Car Selection:** Choose from 3 unique cars with different stats
- **AI Rival Generator:** Each race features a unique AI-generated rival with personality and racing style
- **Race Strategy:** Select from Aggressive, Balanced, or Defensive strategies
- **Dynamic Racing:** Real-time race simulation with smooth animations
- **AI Commentary:** Post-race analysis and rival reactions generated by AI
- **Upgrade Recommendations:** AI suggests car upgrades based on performance

## Getting Started

### Prerequisites

- Node.js 18+
- npm or yarn

### Installation

```bash
# Install dependencies
npm install

# Create .env.local file
cp .env.example .env.local

# Add your API keys
# NEXT_PUBLIC_MIMO_API_KEY=your_mimo_key
# NEXT_PUBLIC_GROQ_API_KEY=your_groq_key
```

### Development

```bash
npm run dev
```

Open [http://localhost:3000](http://localhost:3000) in your browser.

### Build

```bash
npm run build
npm start
```

## Game Flow

1. **Start Screen** → Choose to start a race
2. **Car Selection** → Pick your racing car
3. **Strategy Selection** → Choose your racing strategy
4. **Rival Generation** → AI generates a unique rival (3-5s)
5. **Pre-Race** → View rival info and trash talk
6. **Race Simulation** → 20-second race with real-time animation
7. **Result Screen** → View results, AI commentary, and upgrade recommendations
8. **Repeat** → Race again or return to main menu

## AI Integration

### Fallback Chain

1. **MiMo API** (Primary) - https://token-plan-sgp.xiaomimomo.com/v1
2. **Groq API** (Fallback) - https://api.groq.com/openai/v1
3. **Mock Responses** (Emergency) - Preset templates

### AI Use Cases

- **Rival Generation:** Generate unique names, personalities, backstories, and trash talk
- **Race Commentary:** Analyze race results and generate post-race commentary
- **Upgrade Recommendations:** Suggest car upgrades based on performance
